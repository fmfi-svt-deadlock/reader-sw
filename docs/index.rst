Deadlock Reader (Firmware)
==========================

This is a documentation describing internal design and functioning of embedded firmware of Reader. Reader is a component of project Deadlock_. If you don't know what project Deadlock_ is, start by reading it's documentation.

If you want to understand / modify firmware of the Reader, Reader Firmware Architecture is a good starting point.

.. _Deadlock: https://deadlock.readthedocs.io/
